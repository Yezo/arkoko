# Arkoko

Arkoko is an application that provides a variety of different tools for the online MMORPG "Lost Ark."

## Features

- Marketplace
- Mari's Shop
- Database
- Auction Bid Calculator
- Oreha Crafting Calculator

## Technologies

- React
- TypeScript
- Node & Express
- MongoDB & Mongoose
- Tailwind CSS
- React Router
